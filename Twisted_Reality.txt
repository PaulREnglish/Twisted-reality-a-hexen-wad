* Play Information *
Game : Hexen
Map # : Starts on Map01
Single Player : Designed for 
Cooperative 2-4 Player : No
Deathmatch 2-4 Player : No
Other game styles : none

Base : New from scratch
Build Time : a few years, on and off
Editor(s) used : GZdoom builder and Slade
Known Bugs : (see below)
The rotating polyobject can randomly crash the game

Tested With : GZDoom
